## Setup

1. Install

```bash
npm i
```

2. Run dev server

```bash
npm run dev
```

3. Build and preview

```bash
npm run build && npm run preview
```

## Deploy (GitHub Pages)

Pushing to `main` triggers the GitHub Actions workflow in `.github/workflows/deploy.yml` and publishes to Pages.

Site URL (after first deploy): `https://MrMoody098.github.io/ForMyLove/`
